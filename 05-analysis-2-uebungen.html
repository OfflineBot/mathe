1 · Reihen & Taylor

→ Theorie: Reihen (Theorie)

Aufgabe 1.1 — Drei Reihen prüfen

Untersuche: (a) $\sum \frac{n!}{n^n}$, (b) $\sum \frac{(-1)^n}{\sqrt n}$, (c) $\sum \frac{n^2}{2^n}$.

Lösung:

  1. (a) Quotient: $\frac{(n+1)!/(n+1)^{n+1}}{n!/n^n} = \frac{n^n}{(n+1)^n} = (n/(n+1))^n \to 1/e < 1$. ⇒ konv.
  2. (b) Leibniz: $1/\sqrt n$ alternierend, fallend, $\to 0$. ⇒ konv. Nicht absolut (Zeta mit $s = 1/2$).
  3. (c) Quotient: $\frac{(n+1)^2/2^{n+1}}{n^2/2^n} = \frac{(n+1)^2}{2 n^2} \to 1/2 < 1$. ⇒ konv.
Aufgabe 1.2 — Quotientenkriterium $3^n/n!$

Untersuche $\displaystyle \sum_{n=1}^\infty \frac{3^n}{n!}$ auf Konvergenz.

Lösung: $|a_{n+1}/a_n| = 3/(n+1) \to 0 < 1$. Reihe konvergiert (absolut). Tatsächlich $\sum 3^n/n! = e^3 - 1$ (ab $n=1$).

Aufgabe 1.3 — Wurzelkriterium

Untersuche $\displaystyle \sum_{n=1}^\infty \left(\frac{n}{2n+1}\right)^n$.

Lösung: $\sqrt[n]{|a_n|} = n/(2n+1) \to 1/2 < 1$. Reihe konvergiert.

Aufgabe 1.4 — Leibniz vs. absolute Konvergenz

Konvergiert $\displaystyle \sum_{n=1}^\infty \frac{(-1)^n}{n^2}$ absolut, bedingt oder divergiert?

Lösung: $\sum 1/n^2 = \pi^2/6$ konvergiert (Zeta, $s = 2 > 1$). Also absolut konvergent.

Aufgabe 1.5 — Konvergenzradius

Bestimme Konvergenzradius von $\displaystyle \sum_{n=1}^\infty \frac{(x-2)^n}{n \cdot 3^n}$.

Lösung: $r = \lim |a_n/a_{n+1}| = \lim 3(n+1)/n = 3$. Konv. für $|x-2| < 3$, also $x \in (-1, 5)$.

Aufgabe 1.6 — Taylor-Produkt

Bestimme das Taylor-Polynom Grad 3 von $f(x) = e^x \cos x$ um $x_0 = 0$.

Lösung:

$e^x = 1 + x + x^2/2 + x^3/6 + \dots$, $\cos x = 1 - x^2/2 + \dots$.

Sammle Terme bis Grad 3: $1 + x + 0 \cdot x^2 + (1/6 - 1/2)x^3 = 1 + x - x^3/3$.

2 · Mehrdim. Differentiation

→ Theorie: Mehrdim. Diff. (Theorie)

Aufgabe 2.1 — Gradient + Hesse

$f(x, y) = x^3 y + x y^3$. Bestimme $\nabla f$ und $H_f$.

Lösung:

$\nabla f = (3 x^2 y + y^3,\; x^3 + 3 x y^2)$.

$H_f = \begin{pmatrix} 6 x y & 3 x^2 + 3 y^2 \\ 3 x^2 + 3 y^2 & 6 x y \end{pmatrix}$. Symmetrisch ✓ (Schwarz).

3 · Extrema & Lagrange

→ Theorie: Extrema (Theorie)

Aufgabe 3.1 — Extrema in 2D

Finde + klassifiziere alle Extrema von $f(x, y) = x^2 + xy + y^2 - 6 x + 2$.

Lösung:

$\nabla f = (2x + y - 6,\; x + 2y) = 0$. Aus zweiter: $x = -2y$. In erste: $-4y + y - 6 = 0 \Rightarrow y = -2$, $x = 4$.

$H = \begin{pmatrix} 2 & 1 \\ 1 & 2 \end{pmatrix}$. $\det H = 3 > 0$, $A = 2 > 0$ → striktes lokales (und globales, da konvex) Minimum.

$f^* = 16 - 8 + 4 - 24 + 2 = -10$.

Aufgabe 3.2 — Extrema in 3D

$f(x, y, z) = x^2 + y^2 + z^2 - 2 x + 4 y - 6 z + 1$. Globales Minimum?

Lösung:

Hesse $= 2 I$ pos. def. ⟹ konvex. $\nabla f = (2x - 2, 2y + 4, 2z - 6) = 0 \Rightarrow (1, -2, 3)$.

$f^* = 1 + 4 + 9 - 2 - 8 - 18 + 1 = -13$.

Aufgabe 3.3 — Lagrange mit Ellipse

$\max\; x y$ u.d.N. $x^2 + 4 y^2 = 4$.

Lösung:

$L = xy - \lambda(x^2 + 4y^2 - 4)$. $\partial_x: y = 2\lambda x$, $\partial_y: x = 8\lambda y$.

Aus beiden: $x = 16\lambda^2 x \Rightarrow \lambda^2 = 1/16, \lambda = \pm 1/4$.

NB: $x^2 + 4(2\lambda x)^2 = 2 x^2 = 4 \Rightarrow x = \pm \sqrt 2, y = \pm \sqrt 2/2$.

$f^* = xy = \pm 1$. Maximum $= 1$.

4 · Mehrfachintegrale

→ Theorie: Mehrfachintegrale (Theorie)

Aufgabe 4.1 — Reihenfolge tauschen (Dreieck)

$\iint_D xy\, dA$ über Dreieck mit Ecken $(0,0), (1,0), (0,1)$.

Lösung-A: $\int_0^1 \int_0^{1-x} xy\, dy\, dx = \int_0^1 x \cdot (1-x)^2/2\, dx = (1/2)(1/2 - 2/3 + 1/4) = 1/24$.

Lösung-B: $\int_0^1 \int_0^{1-y} xy\, dx\, dy = 1/24$. ✓

Aufgabe 4.2 — Doppelintegral kartesisch

$\iint_D (x + y)\, dA$ über $D = \{(x, y): 0 \le x \le 2,\; 0 \le y \le x\}$.

Lösung: $\int_0^2 \int_0^x (x + y)\, dy\, dx = \int_0^2 (3x^2/2)\, dx = 4$.

Aufgabe 4.3 — Polarkoordinaten

$\iint_D \sqrt{x^2 + y^2}\, dA$ über Vollkreis $D = \{x^2 + y^2 \le 4\}$.

Lösung: $\int_0^{2\pi} \int_0^2 r \cdot r\, dr\, d\phi = 2\pi \cdot 8/3 = 16\pi/3$.

Aufgabe 4.4 — Kugelkoordinaten (Kugel-Volumen)

Volumen einer Kugel mit Radius $R$.

Lösung: $V = \int_0^{2\pi} \int_0^\pi \int_0^R r^2 \sin\theta\, dr\, d\theta\, d\phi = (R^3/3) \cdot 2 \cdot 2\pi = 4 \pi R^3/3$. ✓

5 · Gewöhnliche DGL

→ Theorie: ODE (Theorie)

Aufgabe 5.1 — ODE trennbar

Löse $y' = x y^2$, $y(0) = 1$.

Lösung:

$dy/y^2 = x\, dx \Rightarrow -1/y = x^2/2 + C$. $y(0) = 1: C = -1$. Also $y = \dfrac{2}{2 - x^2}$.

Definiert auf $(-\sqrt 2, \sqrt 2)$.

Aufgabe 5.2 — ODE linear 1. Ordnung

Löse $y' - 2 y = e^{3 x}$, $y(0) = 0$.

Lösung:

$\mu = e^{-2x}$. $(\mu y)' = e^x \Rightarrow \mu y = e^x + C$. $y = e^{3x} + C e^{2x}$. $y(0) = 0 \Rightarrow C = -1$. $y = e^{3x} - e^{2x}$.

Aufgabe 5.3 — ODE 2. Ordnung mit doppelter Resonanz

$y'' - 4 y' + 4 y = e^{2 x}$, $y(0) = 0$, $y'(0) = 0$.

Lösung:

$\lambda^2 - 4\lambda + 4 = (\lambda - 2)^2 = 0$. Doppelter EW $\lambda = 2$. $y_h = (C_1 + C_2 x) e^{2x}$.

Inhomogenität $e^{2x}$ ist in $y_h$ (sogar doppelt). Ansatz $y_p = A x^2 e^{2x}$.

Einsetzen: $y_p'' - 4y_p' + 4y_p = 2A e^{2x} = e^{2x} \Rightarrow A = 1/2$.

$y(x) = (C_1 + C_2 x + x^2/2) e^{2x}$. AWP: $C_1 = 0$, $C_2 = 0$.

$y(x) = (x^2/2) e^{2x}$.
